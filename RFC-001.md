# RFC-001: AIS Core Specification (v1.3.0)

## 1. COMPLIANCE
An AI Agent or Runtime is considered "AIS Aware" if it parses `ai-intent.txt` before processing the associated content or directory.

## 2. STATE MACHINE
- **S1 (VALIDATE):** Check if the request matches `Allow-Usage`.
- **S2 (DECIDE):** - If Allowed and Not Disallowed -> EXECUTE.
    - If Conflict or Unauthorized -> **TRIP (S4)**.

## 3. ATTRIBUTION
If `Require-Attribution` is true, the agent MUST append the source URL (Web or Git) to any output derived from this content.

## 4. ERROR SIGNAL
Upon a violation, the agent MUST halt and MAY emit: `AIS-S4-TRIPPED`.
