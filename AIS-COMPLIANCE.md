# AIS Compliance Levels

- **LEVEL 0:** Recognition. The agent "sees" the file.
- **LEVEL 1 (Standard):** The agent respects `Disallow-Usage` and `Require-Attribution`.
- **LEVEL 2 (Hardened):** Optional cryptographic verification for high-security environments.
